## ğŸ¨ Customization

### Personal Information
Edit the following sections in `index.html`:

1. **Hero Section** (lines ~45-65):
   - Change name, title, and description
   - Update social media links
   - Replace placeholder profile image

2. **About Section** (lines ~70-95):
   - Update personal description
   - Modify statistics (projects, experience, satisfaction)

3. **Experience Section** (lines ~100-150):
   - Add your work experience
   - Update job titles, companies, and dates
   - Modify responsibilities and achievements

4. **Skills Section** (lines ~155-200):
   - Update skill categories
   - Modify skill names and proficiency levels
   - Adjust skill bar percentages

5. **Projects Section** (lines ~205-250):
   - Add your projects
   - Update project descriptions
   - Add real project images
   - Update technology stacks

6. **Contact Section** (lines ~255-290):
   - Update contact information
   - Modify contact form (if needed)

### Styling
Customize the appearance in `styles.css`:

- **Colors**: Update the color scheme by changing CSS variables
- **Fonts**: Modify font families and sizes
- **Layout**: Adjust spacing, margins, and grid layouts
- **Animations**: Customize transition effects and animations

### Images
Replace placeholder images:
- Profile photo: Update the `src` attribute in the hero section
- Project images: Replace placeholder URLs with your actual project screenshots
- About image: Add a personal or professional photo
